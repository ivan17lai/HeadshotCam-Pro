## Usage
#### 點擊 " 輔助證件照裁切系統.exe " 兩下打開啟動器
#### 從下拉選單選擇你要使用的鏡頭
#### 等待幾一段時間啟動(取決於攝影機啟動速度)
#### 點擊匯入資料載入學生資料
